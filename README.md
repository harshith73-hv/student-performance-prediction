# Student Performance Prediction using Machine Learning

This project predicts student final scores using machine learning techniques. It covers data preprocessing, feature engineering, model training, and performance evaluation.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Features
- Data cleaning and preprocessing
- Regression model training
- Model evaluation using MAE, RMSE, and R²
- Visualization of actual vs predicted scores

## How to Run
```bash
pip install pandas numpy scikit-learn matplotlib seaborn
python student_performance_prediction.py
